Built.io Contentstack - Javascript SDK

Overview

Built.io Contentstack is a content management system that facilitates the process of publication by separating the content from site-related programming and design.

Installation

For Node.js

You need to install node.js v4.8.4 or later to use the Built.io Contentstack SDK.

  1. Run this command in a Terminal or Command Prompt to obtain and install the latest version of the Javascript SDK on your system. 
    $ npm install contentstack
    Note: You might need administrator privileges to perform this installation.
  2. To require the SDK module in your application use the following command. 
    var contentstack = require('contentstack');

For web browsers

  1. Download the latest version of the Built.io Contentstack Javascript SDK from here.
  2. Extract the zip package to get the contentstack.min.js file, and include it in a script tag as shown below. 
    <script type="text/javascript" src="contentstack.min.js"></script>

Initalize the Stack

Built.io Contentstack object would be available for use, as shown below. 

var Stack = Contentstack.Stack({
                "api_key": "blt123456789000000",
                "access_token": "bltqwertyuiop1234",
                "environment": "development"
            });
